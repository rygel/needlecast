## [0.7.4] — 2026-05-01

### Added
- **Workspace import/export** — export your current workspace state and import it later from the File menu.

### Changed
- **Dedicated workspace file extension** — workspace exports now use `.needlecast-workspace` by default.

### Fixed
- **Privacy-mode model compatibility** — migrated project privacy handling to `isPrivate` while preserving compatibility with stored `private` JSON fields.
- **Nested private-project redaction** — Explorer now correctly redacts paths for private projects nested in folders.
- **Claude quota robustness** — removed raw usage payload logging on failures and improved handling of missing usage fields.
