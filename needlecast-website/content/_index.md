Welcome to Needlecast - a modern approach for an agentic coding environment.

Needlecast is a desktop application that provides a comprehensive development environment with integrated terminals, syntax editors, project organization, and AI agent support.

## Key Features

* **Project Tree & Organization** - Manage your projects efficiently
* **Auto-detected Commands** - Intelligent command discovery
* **Embedded Terminals** - Multiple terminals within the app
* **Syntax Editor & File Explorer** - Full-featured code editing
* **Image & SVG Viewer** - Visual file support
* **Log Viewer Panel** - Centralized logging
* **Renovate Integration** - Automated dependency updates
* **Prompt & Command Libraries** - Reusable AI prompts and commands
